\doxysubsection{Detailed Description}
Singleton that keeps track of the game state, provides the utility to (re)start the game and serves as a registry for other Game\+Objects that carry out management tasks like the \mbox{\hyperlink{class_escape_reality_1_1_time_tracker}{Time\+Tracker}}.
